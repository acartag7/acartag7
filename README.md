### What I'm building

**[Edictum](https://github.com/edictum-ai/edictum)** — open-source runtime governance library for AI agent tool calls. Enforces deterministic safety contracts (preconditions, postconditions, session limits) at the tool-call boundary with a full audit trail.

Multi-language SDKs: [Python](https://github.com/edictum-ai/edictum) (⭐24, MIT, PyPI `edictum`) · [TypeScript](https://github.com/edictum-ai/edictum-ts) · [Go](https://github.com/edictum-ai/edictum-go) · [OpenClaw plugin](https://github.com/edictum-ai/edictum-openclaw).

Hosted platform at [edictum.ai](https://edictum.ai) — approvals, audit feeds, fleet monitoring.
